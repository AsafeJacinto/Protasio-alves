Qual a diferença entre Front-end e Back-end?

Front-End

Chamamos de Front-End tudo aquilo com que o usuário interage diretamente, como botões, menus e textos.

  • HTML

    • É onde estruturamos a página.

  • CSS

    • Define o estilo da página, como cores, fontes e posições dos elementos em geral.

  • JavaScript

    • Adiciona dinamismo à página, permitindo funções mais complexas para o usuário.



    Back-End

    Chamamos de Back-End tudo aquilo que o usuário não vê, mas que é responsável pelo funcionamento esperado da interface.

    Inclui linguagens de programação, bancos de dados e APIs.

  • Python

    • Linguagem de programação conhecida pela sua simplicidade e versatilidade.

  • PHP

    • Linguagem de programação popular para desenvolvimento web.

  • SQL

    • Linguagem de programação voltada para manipulação de dados.
